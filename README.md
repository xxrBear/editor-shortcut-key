### 代码编辑快捷键

- `CTRL+ALT+L`:  代码格式标准化
- `CTRL+D`:      向下复制当前行
- `CTRL+X`:      剪接当前行
- `CTRL+Y`:      删除当前行
- `CTRL+SHIFT+U`:当前代码全部大写
- `CTRL+/`:      注释一行代码

### 搜索/替换快捷键
- `CTRL+F`:      当前文本搜索
- `CTRL+SHIFT+F`: 全局搜索
- `CTRL+R`: 当前文件替换文本
- `CTRL+SHIFT+R`: 全局替换文本
